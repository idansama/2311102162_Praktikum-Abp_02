* **MaterialApp** → Widget utama aplikasi Flutter.
* **Scaffold** → Struktur dasar halaman aplikasi.
* **AppBar** → Bar bagian atas aplikasi.
* **Container** → Kotak untuk mengatur tampilan widget.
* **Column** → Menyusun widget secara vertikal.
* **Text** → Menampilkan teks.
* **GridView** → Menampilkan item dalam bentuk grid.
* **ListView** → Menampilkan daftar item secara vertikal.
* **ListView.builder** → Membuat list secara dinamis dari data.
* **ListView.separated** → List dengan garis pemisah antar item.
* **ListTile** → Item daftar dengan icon dan teks.
* **Card** → Tampilan kartu dengan bayangan.
* **Stack** → Menumpuk widget.
* **Positioned** → Mengatur posisi widget dalam Stack.
* **SingleChildScrollView** → Membuat halaman bisa di-scroll.
* **SizedBox** → Memberi jarak atau ukuran kosong.
